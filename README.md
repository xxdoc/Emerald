# Emerald
面向游戏的轻量级绘图框架

# 支持

# 功能
√ 基本图形的绘制  
√ 即时控件  
√ 安全游戏存档  
√ 游戏资源管理器  
√ 游戏页面管理器  
√ 封装Bass的音乐播放器  
√ 动画系统
